В папке DNN/Weights хранятся папки с историей обучения нейронной сети
Имя папки соответствует дате и времени обучения модели нейронной сети.

ВЫЖНО!!!!
Модель нейронной сети должна четко соответствовать модели, при которой были  сохранены выбранные веса.

------------------------------------------------------------------------------------------------------------------------

В папке DNN/Transformers хранятся заполненные трансформеры (нормализаторы).
Имя файла соответствует дате и времени обучения модели нейронной сети.
Для корректной работы модели нейронной сети необходимо нормализировать значения, подаваемые на вход.

ВЫЖНО!!!!
Выходные значения для регрессии нормализировать не нужно.
